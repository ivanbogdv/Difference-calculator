# Difference calculator 

### Description
The difference calculator is a program that determines the difference between two data structures.
```bash
{
  - follow: false
    host: hexlet.io
  - proxy: 123.234.53.22
  - timeout: 50
  + timeout: 20
  + verbose: true
}
```

### Install
```bash
git clone git@github.com:ivanbogdv/frontend-project-44.git
make install 
make lint 
```

### Supported formats
```bash
json
yaml
yml
```

### Program report form
```bash
stylish
plain
json
```

### Instructions
```bash
gendiff -f format filepath1 filepath2 
gendiff -h
```
